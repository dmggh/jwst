Get Best References lets you interactively explore best reference selections made by CRDS given particular dataset parameters.

Instrument Context defines the context for which best references will be determined. It defines the parameter choices which will be available as well as the reference mappings which will be used to infer best references.

Only instrument contexts already known to CRDS can be specified. {% endblock help %}
