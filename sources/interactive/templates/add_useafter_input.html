Add UseAfter derives a new reference mapping from an old one by inserting a new useafter clause into the specified match case. A new reference mapping will be automatically generated and submitted.

Old Mapping is the reference mapping to which the useafter clause will be added, producing a new mapping.

Match Tuple is a set of parameters which a dataset must match for the new useafter clause to be applicable.

UseAfter Date is the date after which the new reference file is applicable.

UseAfter File is the filename of the reference file being added.
