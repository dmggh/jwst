Batch Submit References supports uploading a list of reference files for a single instrument and reference type, i.e. one rmap. After checking and storing the reference files, new CRDS mapping files which refer to them are generated.

Old Context to Derive From The name of the pipeline context which will be recursively updated to insert the new reference files.

Change Level The degree to which the new files are expected to impact science results.

Description information about what's new in this file and what the expected impacts are.

Creator author of the contents or changes to the file, not necessarily the submitter. The submitter is known based on login information.

Submitted References Uploaded Files lets you upload a list of new references to CRDS. Each reference file must certify or the entire batch will be rejected. If you are a member of group crdsoper, you can make your own subdirectory in /grp/crds/{{observatory}}/ingest and copy your submitted files there, then refresh this page and select Server Directory and the directory you created from the drop-down menu. {% include "auto_rename_help.html" %} {% endblock help %}
