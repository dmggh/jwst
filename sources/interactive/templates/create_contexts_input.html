Create Parent Contexts automatically generates and submits new pipeline and instrument contexts which refer to your new rmaps.

Pipeline defines the starting context which will be updated to refer to new rmaps. The pipeline context refers to a set of existing instrument contexts.

Files you list in the New Rmaps box will be substituted into copies of the existing instrument contexts generating new ones. Updated Rmaps can be listed one file per line, separated by spaces, or seperated by commas.

Description should document the general rationale or purpose behind the set of rmap updates. {% endblock help %}
