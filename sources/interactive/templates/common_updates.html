Create Parent Contexts generates and submits new pipeline and instrument mappings which refer to new reference mappings you created elsewhere.

Replace Reference In Rmap will substitute a single reference file for another in an existing reference mapping.

Add New UseAfter lets you add a new useafter date and corresponding reference file to an existing reference mapping.

In all cases, the mappings and reference files must already have been submitted to CRDS. These features let you tweak existing reference maps or automatically generate a new parent context for new reference maps. {% endblock help %}
